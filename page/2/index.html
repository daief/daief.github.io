JavaScript 中的面向对象

发表于：2020/02/23
JavaScript

在 JavaScript 中没有类的概念，因此它的对象与基于类的语言中的对象不同。

ECMA-262 把对象定义为：无序属性的集合，其属性可以包含基本值、对象或函数。

一直以来在编程中都是使用 class、extends 关键字，而这实际上是 ES6 中添加的语法糖。在熟练使用语法糖的同时，有必要了解其背后的原生实现。
